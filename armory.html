Обычный выстрел

  • Требования: Ничего
  • Очки Мощи: Ничего
Описание:

Персонаж делает обычный выстрел из дальнобойного оружия. Ничего особенного, просто и эффективно.

Система:

При применении обычного Выстрела персонажа делает проверку по Ловкость + Стрельба.

  • Проверка: Ловкость + Стрельба
  • Стоимость: Ничего
  • Инициатива: +1
  • Сложность: Стандарт
  • Точность: +0
  • Урон: Урон оружия*
  • Передвижение: +0

Длинная Очередь

  • Требования: Сила 2, Стрельба 2
  • Очки Мощи: Ничего
Описание:

Персонаж выпускает в цель весь магазин полевая врагов градом из свинца

Система:

Персонаж проходит одну проверку атаки, но её пул увеличивается на десять d10, а сложность возрастает на два пункта из‑за сильной отдачи. При этом он не выбирает одну конкретную цель, а скорее выбирает направление стрельбы, проводя линию в три гекса шириной. Таким образом в зону огня может попасть больше одной цели. За каждый дополнительный упех на попадание после первого, персонаж может выбрать дополнительных жертв в линии стрельбы и на этапе проверки урона разделить пулл урона между жертвами. Дополнительные успехи увеличивают пул проверки урона, как если бы это был простой одиночный выстрел. Персонаж, который ведёт огонь длинными очередями, не может целиться в определённую часть тела жертвы. Персонаж может выпустить длинную очередь, только если магазин его оружия полон хотя бы на половину.

  • Проверка: Ловкость + Стрельба
  • Стоимость: Ничего
  • Инициатива: -2
  • Сложность: +2
  • Точность: +10
  • Урон: Урон оружия*
  • Передвижение: -2

Короткая очередь

  • Требования: Стрельба 2
  • Очки Мощи: Ничего
Описание:

Персонаж стреляет короткими очередями по три пули за раз

Система:

Персонаж тратит три патрона за одну атаку, но взамен увеличивает пул проверки атаки на два d10. Не все виды оружия имеют такой режим стрельбы. Сложность проверки атаки возрастает на один пункт из‑за отдачи. До полнительные успехи увеличивают пул проверки урона, как если бы это был простой одиночный выстрел.

  • Проверка: Ловкость + Стрельба
  • Стоимость: Ничего
  • Инициатива: +0
  • Сложность: +1
  • Точность: +2
  • Урон: Урон оружия*
  • Передвижение: +0

Прицеливание

  • Требования: Стрельба 1
  • Очки Мощи: Ничего
Описание:

Персонаж концентрируется на цели, готовясь к точному выстрелу.

Система:

Персонаж тратит полный ход на наведение оружия на конкретную цель при этом делает проверку на Внимательность + Стрельба. Каждый успех добавляет +1 к Точности следующего выстрела. Бонус к Точности не может привешать Внимательность персонажа. Персонаж не может целиться в цель, которая прошла больше чем Смекалка * 2 персонажа гексов в ход. Сложность может меняться в зависимости от условий стрельбы.

  • Проверка: Внимательность + Стрельба
  • Стоимость: Ничего
  • Инициатива: +0
  • Сложность: +0
  • Точность: Ничего
  • Урон: Ничего
  • Передвижение: 1

Быстрая грязная математика

Вкратце, вот формулы для определения пула кубиков для каждой стадии атаки. Модифицируйте их по своему вкусу. Не волнуйтесь, если они кажутся немного пугающими: с практикой вычисление пула кубов будет так же естественно, как дышать. Помните, что независимо от модификаций, минимальный пул кубиков для каждого из этих бросков – один куб. Всегда можно попробовать.

  • Инициатива:

    1д10 + (Ловкость + Смекалка) + модификатор маневра + другие модификаторы

  • Точность:

    Ловкость + Способности* + модификация маневра - пенальти от Уровней Здоровья = пул кубов.

    Способности:

    1. Атлетика для метательного оружия
    2. Драка для рукопашных ударов без холодного оружия (дубины, ножы, бензопилы и т.п.)
    3. Уклонение для защитных действий
    4. Рукопашный бой для холодного оружия
    5. Стрельба для дальнобойного оружия

  • Урон:

    Урон оружия* + кол-во успехов на попадание = пул кубов.

    >Возможность поглощения урона наступает только после того, как персонаж нанесет урон.

    Урон оружия*: Урон оружия необходимо проверять непосредственно в параметрах самого оружия. Учитывайте все особенности калибра и дополнительные модификаторы при совершении любого действия, связанного с данным оружием.

  • Перемещение:

    Ловкость + модификация маневра + другие модификаторы = количество гексов, перемещаемых за ход.
