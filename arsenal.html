Двуствольное ружьё / Обрез

Описание:

Двуствольное ружьё 12 калибра — это не игрушка. Два ствола, два выстрела — два шанса закончить дело, прежде чем оно обернётся для тебя плохо. Ему плевать, кто ты: смертный, Сородич или что-то похуже. Когда оно заговорит, слушают все. Дробь разлетается, как шёпот по ночной улице — на ближней дистанции промахнуться невозможно. Да, оно громкое, да, оно оставляет кровавый беспорядок и привлекает лишние глаза. Но иногда... тебе это и нужно. Только не тяни с перезарядкой — не успеешь моргнуть, как пожалеешь об этом.

Модификация:

Иногда тебе не нужен весь этот длинный ствол. Укороти его — и получишь обрез. Злобный, компактный, удобно спрятать под пальто. Да, дальность пострадает, но на ближней дистанции разброс дроби — просто смерть. Если всё же решишь модифицировать оружие, тогда смотри на цифры после знака /."

Быстрая грязная математика

Вкратце, вот формулы для определения пула кубиков для каждой стадии атаки. Модифицируйте их по своему вкусу. Не волнуйтесь, если они кажутся немного пугающими: с практикой вычисление пула кубов будет так же естественно, как дышать. Помните, что независимо от модификаций, минимальный пул кубиков для каждого из этих бросков – один куб. Всегда можно попробовать.

  • Инициатива:

    1д10 + (Ловкость + Смекалка) + модификатор маневра + другие модификаторы

  • Точность:

    Ловкость + Способности* + модификация маневра - пенальти от Уровней Здоровья = пул кубов.

    Способности:

    1. Атлетика для метательного оружия
    2. Драка для рукопашных ударов без холодного оружия (дубины, ножы, бензопилы и т.п.)
    3. Уклонение для защитных действий
    4. Рукопашный бой для холодного оружия
    5. Стрельба для дальнобойного оружия

  • Урон:

    Урона от Калибра Оружия* + кол-во успехов на попадание = пул кубов.

    >Возможность поглощения урона наступает только после того, как персонаж нанесет урон.

  • Перемещение:

    Ловкость + модификация маневра + другие модификаторы = количество гексов, перемещаемых за ход.
