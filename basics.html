Блок

  • Требования: Ничего
  • Очки Мощи: Ничего

Описание:

Базовый защитный прием, представляющий из себя преграждение доступа атаке оппонента к уязвимым точкам тела персонажа, обычно кистью или предплечьем.

Система:

При использовании персонажем Блока, значение Уклонения добавляется к Выносливости на броске поглощения урона. Вместе с тем, некоторые атаки, например, Захват, игнорируют бонус Блока, посему урон, нанесенный ими, может быть поглощен только Выносливостью персонажа.

Блок создает и два иных эффекта. Первый - на протяжении хода после использования персонажем Блока, следующий после Блока прием получает +2 к Инициативе. Это отражает особенность правильно поставленного блока - быть стартовым положением для большей части рукопашных ударов. Таким образом персонаж может контратаковать из защитной стойки быстрее, чем из положения завершения атакующего приема. Второй - Блок не допускает отправку персонажа в Нокдаун любой атакой, за исключением атак, игнорирующих Блок, например, Захвата.

Блоком нельзя заблокировать атаку с использованием оружия. Для этого персонаж должен использовать специальный Продвинутый Прием или, с использованием оружия, Оружейный Прием. Блок может быть разыгран, как Прием Отмена.

  • Проверка: Выносливость + Уворот на понижание урона
  • Стоимость: Ничего
  • Инициатива: +4
  • Сложность: Стандарт
  • Точность: Ничего
  • Урон: Ничего
  • Передвижение: +0

Уклонение

  • Требования: Ничего
  • Очки Мощи: Ничего

Описание:

Бесценное искусство ухода с линии атаки.

Система:

При применении Уклонения персонажем, делается бросок Ловкость + Уклонение. Каждый, полученный в этом броске успех, отнимает один успех от броска попадания противника.

Сложность Уклонения растет прямо пропорциональна количеству противников. Если на персонажа направлено несколько атак он может выбрать, от какой атаки уклониться или разделить свой запас костей.

  • Проверка: Ловкость + Уворот
  • Стоимость: Ничего
  • Инициатива: +6
  • Сложность: Стандарт
  • Точность: Ничего
  • Урон: Ничего
  • Передвижение: +1

Захват

  • Требования: Ничего
  • Очки Мощи: Ничего

Описание:

Захват — грубый, но эффективный манёвр. Этот приём позволяет не убить противника, а сломить его сопротивление и удержать в железной хватке. Манёвр делится на два режима: простое удержание для обездвиживания цели и клинч, когда дело доходит до грубой физической расправы

Система:

Простое удержание: Захватывающий стремительно лишает противника свободы движения, блокируя его руками или всей массой тела. Цель не может предпринимать никаких действий, кроме как отчаянно пытаться вырваться из захвата. Для этого она проходит встречную проверку Сила + Драка против Сила + Драка захватывающего.

  • Успех: Цель освобождается и может действовать как обычно.
  • Провал: Цель остаётся в плену хватки до следующего хода.

Клинч: Когда хватка превращается в борьбу, оба бойца вступают в безжалостный клинч. Это не аккуратное боевое искусство, а дикая схватка с единственной целью — нанести как можно больше урона.

  • Каждый ход персонажи действуют согласно своей инициативе.
  • Урон наносится автоматически, равный Силе, без необходимости подтверждать попадание.
  • Разорвать клинч можно только через встречную проверку Сила + Драка против противника.
  • Сила + Драка
  • Стоимость: Ничего
  • Инициатива: -1
  • Сложность: Стандарт
  • Точность: +0
  • Урон: +0
  • Передвижение: +0

Быстрая грязная математика

Вкратце, вот формулы для определения пула кубиков для каждой стадии атаки. Модифицируйте их по своему вкусу. Не волнуйтесь, если они кажутся немного пугающими: с практикой вычисление пула кубов будет так же естественно, как дышать. Помните, что независимо от модификаций, минимальный пул кубиков для каждого из этих бросков – один куб. Всегда можно попробовать.

  • Инициатива:

    1д10 + (Ловкость + Смекалка) + модификатор маневра + другие модификаторы

  • Точность:

    Ловкость + Способности* + модификация маневра - пенальти от Уровней Здоровья = пул кубов.

    Способности:

    1. Атлетика для метательного оружия
    2. Драка для рукопашных ударов без холодного оружия (дубины, ножы, бензопилы и т.п.)
    3. Уклонение для защитных действий
    4. Рукопашный бой для холодного оружия
    5. Стрельба для дальнобойного оружия

  • Урон:

    Сила + модификатор = пул кубов.

    Возможность поглощения урона наступает только после того, как персонаж нанесет урон. Некоторые Рассказчики могут разрешить персонажу добавить Драку или Фехтование, в зависимости от ситуации, к пулу кубов урона. Это влияет на способность искусного бойца наносить смертельные удары, но может сделать бой исключительно смертельным или несбалансированным. Если вы решите использовать эту опцию, мы рекомендуем вам вычитать кубы прочности противника непосредственно из пула кубов, чтобы не бросать неприлично огромные горсти при каждом ударе.

  • Перемещение:

    Ловкость + модификация маневра + другие модификаторы = количество гексов, перемещаемых за ход.
