Быстрая грязная математика

Вкратце, вот формулы для определения пула кубиков для каждой стадии атаки. Модифицируйте их по своему вкусу. Не волнуйтесь, если они кажутся немного пугающими: с практикой вычисление пула кубов будет так же естественно, как дышать. Помните, что независимо от модификаций, минимальный пул кубиков для каждого из этих бросков – один куб. Всегда можно попробовать.

  • Инициатива:

    Смекалка + бдительность + модификатор маневра, если есть.

  • Точность:

    Ловкость + способности (Атлетика, Драка, Уклонение или Рукопашный бой) + модификация маневра - пенальти от Уровней Здоровья = пул кубов.

  • Урон:

    Сила + модификатор = пул кубов. Возможность поглощения урона наступает только после того, как персонаж нанесет урон. Некоторые Рассказчики могут разрешить персонажу добавить Драку или Фехтование, в зависимости от ситуации, к пулу кубов урона. Это влияет на способность искусного бойца наносить смертельные удары, но может сделать бой исключительно смертельным или несбалансированным. Если вы решите использовать эту опцию, мы рекомендуем вам вычитать кубы прочности противника непосредственно из пула кубов, чтобы не бросать неприлично огромные горсти при каждом ударе.

  • Перемещение:

    Ловкость + модификация маневра = количество гексов, перемещаемых за ход.

Блок

  • Требования: Ничего
  • Очки Мощи: Ничего

Описание:

Базовый защитный прием, представляющий из себя преграждение доступа атаке оппонента к уязвимым точкам тела персонажа, обычно кистью или предплечьем.

Система:

При использовании персонажем Блока, значение Уклонения добавляется к Выносливости на броске поглощения урона. Вместе с тем, некоторые атаки, например, Захват, игнорируют бонус Блока, посему урон, нанесенный ими, может быть поглощен только Выносливостью персонажа.

Блок создает и два иных эффекта. Первый - на протяжении хода после использования персонажем Блока, следующий после Блока прием получает +2 к Инициативе. Это отражает особенность правильно поставленного блока - быть стартовым положением для большей части рукопашных ударов. Таким образом персонаж может контратаковать из защитной стойки быстрее, чем из положения завершения атакующего приема. Второй - Блок не допускает отправку персонажа в Нокдаун любой атакой, за исключением атак, игнорирующих Блок, например, Захвата.

Блоком нельзя заблокировать атаку с использованием оружия. Для этого персонаж должен использовать специальный Продвинутый Прием или, с использованием оружия, Оружейный Прием. Блок может быть разыгран, как Прием Отмена.

  • Стоимость: Ничего
  • Инициатива: +4
  • Точность: Ничего
  • Урон: Ничего
  • Передвижение: +0

Уклонение

  • Требования: Ничего
  • Очки Мощи: Ничего

Описание:

Бесценное искусство ухода с линии атаки.

Система:

При применении Уклонения персонажем, делается бросок Ловкость + Уклонение. Каждый, полученный в этом броске успех, отнимает один успех от броска попадания противника.

Сложность Уклонения растет прямо пропорциональна количеству противников. Если на персонажа направлено несколько атак он может выбрать, от какой атаки уклониться или разделить свой запас костей.

  • Стоимость: Ничего
  • Инициатива: +6
  • Точность: Ничего
  • Урон: Ничего
  • Передвижение: +1
